1. Filter the data to include passengers only. Calculate the joint distribution (cross-tab) between a passenger’s class and where they embarked.

# Code here

#assign each class type to passenger or crew
my_dict = {
    "1st": "passenger",
    "2nd": "passenger",
    "3rd": "passenger",
    "victualling crew": "crew",
    "engineering crew": "crew",
    "deck crew": "crew"
}

df_titanic["type"] = df_titanic["class"].map(my_dict)
df_titanic

#filter only passengers
df_passengers = df_titanic[df_titanic["type"] == "passenger"]
df_passengers

#calculate joint distribution b/w passenger class and where they embarked
pd.crosstab(df_passengers["class"], df_passengers["embarked"], normalize=True)

3. Make a visualization showing the distribution of a passenger’s class, given where they embarked.

Discuss the pros and cons of using this visualization versus the distributions you calculated before, to answer the previous questions.

<Figure Size: (640 x 480)>

This bar graph shows a clear distinction between the distribution of classes at different embarkment stations for passengers. We can easily see that station B is not a common place for titanic passengers to embark from, and that station S is the most common for passengers to embark from, especially from the 3rd class. Some pros to this visualization that it is very easy to see the differences for each class per station at a quick glance. However, a con is that we don’t know the specific count of passengers per class at each station unless we label them on the plot. A two way table and the distributions calculated will tell you the precise proportion of passengers per class at each station, regardless of how large or small the proportion is. Sometimes the proportions are so similar that the bar heights look almost identical. However, the distributions calculated from earlier will show the precise difference between the proportions.
